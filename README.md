# fish functions

## General Stuff
### Step down 1/2/3 directories
```shell
> ..
> ....
> ......
```

## Git Stuff
### Add files
```shell
> add [ARGS]
```

### Amend last commit with message
```shell
> amend [MESSAGE]
```

### Commit
```shell
> c  [MESSAGE]
> cm [MESSAGE]
```

### Commit all
```shell
> ca [MESSAGE]
```

### Show diff to last commit
```shell
> diffc
```

### Fetch origin
```shell
> fetch
```

### Push to upstream
```shell
> push
> fpush # force-push
```
