## Git Stuff
### Add files
```
> add [ARGS]
```

### Amend last commit with message
```
> amend [MESSAGE]
```

### Commit
```
> c  [MESSAGE]
> cm [MESSAGE]
```

### Commit all
```
> ca [MESSAGE]
```

### Show diff to last commit
```
> diffc
```

### Fetch origin
```
> fetch
```

### Push to upstream
```
> push
> fpush # force-push
```
